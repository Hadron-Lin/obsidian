# 逆用无穷小

$$
\begin{align}
\lim_{ x \to 1 }x-1=\lim_{ x \to 1 } \ln x 
\end{align}
$$
## 理论基础
$$
\begin{align}
\lim_{ x \to 1 } x-1=\lim_{ x \to 1} e^{ \ln x }-e^{ 0 }=\lim_{ x \to 1 }\lim_{ \xi \to 0 }  e^{ \xi }(\ln x)=\lim_{ x \to 1 } \ln x 
\end{align}
$$
