# 二元泰勒展开
$$
\begin{align}
f(x+h,y+k)=f(x,y)+\left( h \frac{ \partial  }{ \partial x }+k\frac{ \partial  }{ \partial y }   \right)\cdot f(x,y)+\frac{1}{2!}\left( h\frac{ \partial  }{ \partial x }  +k\frac{ \partial  }{ \partial y } \right)^2\cdot f(x,y)+\cdots \\

\end{align}
$$
