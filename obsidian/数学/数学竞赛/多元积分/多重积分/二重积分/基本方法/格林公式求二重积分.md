# 格林公式
$$
\begin{align}
\oint Pdx+Qdy=\iint \frac{ \partial Q }{ \partial x } -\frac{ \partial P }{ \partial y } dxdy \\
\oint Pdx=-\iint \frac{ \partial P }{ \partial y }dxdy 
\end{align}
$$
由此可见,二重积分可以转化成闭合曲线积分
