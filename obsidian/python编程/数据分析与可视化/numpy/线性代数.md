# 矩阵乘法

```python
import numpy as np
a=np.ones([2,4])
print(a)

b=np.full([4,3],4)
print(b)

c=np.matmul(a,b)
d=a@b
print(c)
print(d)
```

# 行列式
```python
e=np.arange(1,17)
e=e.reshape([4,4])
print(e)
print(np.linalg.det(e))
```
